# **Escribir en la terminal**

**Para descargar tu carpeta de herramientas:**

git clone https://github.com/maxcollo82/installare.git

Esto creará una carpeta llamada installare con todo lo que hayas subido.

**Para ejecutar tu script directamente:**

cd installare
bash tu_script.sh

<br>

Si tu script de Bash tiene comandos que necesitan permisos (como instalar programas), recuerda que en el PC del cliente tendrás que usar sudo delante de los comandos o del propio script:

sudo bash tu_script.sh
